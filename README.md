EdTech Platform

A full-stack collaborative learning platform built with the MERN stack that enables students to learn, collaborate, and organize their academic resources in one place. The platform provides real-time communication, secure authentication, cloud-based document management, and personalized learning features to enhance the overall learning experience.

Key Features
📚 Share and manage study notes
👥 Create and join study groups
💬 Real-time group chat and doubt resolution using Socket.io
🔐 Secure JWT-based authentication and role-based access control
☁️ Cloud-based file upload and document management with Cloudinary
📊 Personalized learning dashboard with progress tracking
📝 Topic-based discussion forums
🤖 AI-powered quiz generation for self-assessment
